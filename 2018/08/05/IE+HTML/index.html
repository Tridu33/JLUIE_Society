前端网页Workshop总结

前端网页Workshop总结

首先，编程语言是连接机器01语言和人类语言的中间“翻译”语言，解决实际问题。编程语言实现途径有解释型，编译型和脚本型。

 编译型，用专用的编译器，将某种高级语言源代码一次性翻译成可被硬件平台直接运行的二进制机器码。解释型，用专门解释器对源程序逐行解释成特定平台的机器码并立即执行的语言。脚本型，为了缩短传统编译型语言的「编写、编译、链接、运行」过程而创建的计算机编程语言。一个脚本通常是解释运行而非编译。

当然，上面三种分类并不是非此即彼的关系，比如：Python是解释型同时是脚本型，Java既可以说是编译型也可以说是解释型语言，只能说三种编程语言是把人类语言转化为机器语言的几种主要的实现途径，并且可以据此对编程语言进行粗略的分类。第二张PPT01机器语言翻译成人类语言的实现过程，首先设置对机器语言解释的一种约定方式，然后通过约定对代码进行提炼，打包，整合，进而成为高级编程语言，如C语言。

 强调编程语言只是工具，只是一种约定，强调编程过程中重要的是算法数据结构和对现实问题的反向建模的思路，每一种语言对解决实际问题的采取的不同思路，在不同情境下有其活跃的地方。根据不同人对解决问题的不同思路，出现了多种形式的编程语言。
